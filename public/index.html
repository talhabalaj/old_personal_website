Tenda N301 Router Module for Python

A Module for Python to Programmtically Control Tenda Routers (Model N301). It may work with other Tenda Routers having the same Web Interface. Tested with firmware version V12.01.01.42_multi of the Router and I have added functions like get_online_devices(), get_blacklist() and block_device('<mac_adress>'). More functionality will be added later.
